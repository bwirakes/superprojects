# Super Projects

This is the official website for **Super Projects**, a public art organization based in Singapore. 

## About 
Super Projects is dedicated to organizing, curating, and presenting public art initiatives that engage the community and transform public spaces.

## Development

This project is built with [Next.js](https://nextjs.org) and styled with Tailwind CSS. It uses [Bun](https://bun.sh) as the package manager.

### Getting Started

First, install the dependencies using Bun:

```bash
bun install
```

Then, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This website is automatically deployed via Vercel when changes are pushed to the main repository.
